# Proyecto
<div align="justify">
Este <strong>Proyecto Académico</strong> fue desarrollado por el grupo de estudiantes <em><strong>ArquiTechLab</strong></em> de Arquitectura de la <strong>Universidad La América</strong>, en colaboración con la <strong>Corporación de Educación del Norte del Tolima (COREDUCACIÓN)</strong>.
La iniciativa busca renovar y optimizar la infraestructura educativa para ampliar la capacidad estudiantil, mejorar el aprovechamiento del espacio y preservar un enfoque de sostenibilidad, con el propósito de fortalecer y proyectar a la institución como un referente académico y urbano.
</div>
