# Proyecto
Este es un **Proyecto Académico** creado por **_ArquiTechLab_** para La Corporación de Educación del Norte del Tolima (COREDUCACIÓN).


Este **Proyecto Académico** fue desarrollado por el grupo de estudiantes **_ArquiTechLab_** de Arquitectura de la **Universidad La América**, en colaboración con la **Corporación de Educación del Norte del Tolima (COREDUCACIÓN)**.  
La iniciativa busca renovar y optimizar la infraestructura educativa para ampliar la capacidad estudiantil, mejorar el aprovechamiento del espacio y preservar un enfoque de sostenibilidad, con el propósito de fortalecer y proyectar a la institución como un referente académico y urbano.
